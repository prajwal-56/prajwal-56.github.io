carapace

C

A container runtime written in C — basically a mini Docker. It isolates a running process using Linux namespaces and cgroups, giving it its own filesystem, PID space, and network, just like the real thing. Built to actually understand what Docker does under the hood.

A local-network file sharing webapp built with FastAPI. Host it on any machine, and anyone on the same network can hit the URL to upload or download files stored on that device. No cloud, no accounts — just run it and share.
